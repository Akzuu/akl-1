# Install
```
npm install -g gulp
npm install -g bower
npm install
bower install
mvn install
cp .akl.properties ~/
```
# Run
```
mvn spring-boot:run &
gulp serve &
```
or use install.sh


# Private configs
Put all private configs to ~/.akl.properties file.
